## 题目描述

```
371. 两整数之和
给你两个整数 a 和 b ，不使用 运算符 + 和 - ​​​​​​​，计算并返回两整数之和。

示例 1：

输入：a = 1, b = 2
输出：3
示例 2：

输入：a = 2, b = 3
输出：5


提示：

-1000 <= a, b <= 1000

```

## 思路

使用位运算来模拟整数加法以及进位

```
1 & 1 = 1
0 & 0 = 0
1 ^ 1 = 0
1 ^ 0 = 1
0 ^ 0 = 0
```

由上可以得到

1. a ^ b 等于 a + b 不考虑进位的数
2. (a & b) 等于 a + b 的进位

### 代码

不难写出以下代码

```js
/**
 * @param {number} a
 * @param {number} b
 * @return {number}
 */
var getSum = function (a, b) {
  while (b !== 0) {
    let carry = (a & b) << 1;
    a = a ^ b;
    b = carry;
  }
  return a;
};
```

- 时间复杂度: $O(log(Math.max(a, b)))$
- 空间复杂度: $O(1)$

递归版本

```js
/**
 * @param {number} a
 * @param {number} b
 * @return {number}
 */
var getSum = function (a, b) {
  return b === 0 ? a : getSum(a ^ b, (a & b) << 1);
};
```
